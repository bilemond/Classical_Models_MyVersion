湖上三千里，山中一夜风。   
光辉一夜雨，日暮一声天。   
秋草无人见，春风入夜深。   
月明千里月，风起一声钟。   
两地无人见，无人更別离。   
相思不可见，此地有谁家。   
和雨寒来雨，寒山夜夜声。
